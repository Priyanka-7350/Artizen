# Artizen Café — Website

A premium, single-page marketing website for **Artizen Café**, an art café in Pune. Tagline: **"Eat • Sip • Paint"**.

## Quick Start

| Layer | How to run |
|---|---|
| **Frontend** | `cd frontend && yarn install && yarn start` |
| **Backend** | `cd backend && pip install -r requirements.txt && uvicorn server:app --reload` |
